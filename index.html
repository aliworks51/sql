Exemples Concrets

Création d'une Table Utilisateurs

CREATE TABLE utilisateurs ( id INT AUTO_INCREMENT PRIMARY KEY, nom VARCHAR(50) NOT NULL, email VARCHAR(100) UNIQUE NOT NULL, date_inscription TIMESTAMP DEFAULT CURRENT_TIMESTAMP, actif BOOLEAN DEFAULT 1 );

Requêtes de Base

Opération Exemple
